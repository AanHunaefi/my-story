# Aan Hunaefi — Website Package

Isi folder ini adalah website statis sederhana (HTML/CSS/JS) untuk portfolio penulis dan novel "Sebelum Dunia Menjadi Sunyi".

## File penting
- `index.html` — halaman utama
- `styles.css` — styling
- `script.js` — interaksi kecil
- `sitemap.xml` — sitemap (sesuaikan `loc` dengan domainmu)
- `cover-placeholder.jpg` — placeholder cover (ganti dengan cover asli)

## Cara deploy cepat

### GitHub Pages (gratis)
1. Buat repo baru di GitHub.
2. Upload semua file (atau push via Git).
3. Buka _Settings > Pages_, pilih branch `main` dan folder `/root`.
4. Save — halaman akan tersedia di `https://username.github.io/repo-name/`.

### Netlify (drag & drop)
1. Daftar/masuk ke netlify.com.
2. Drag & drop folder ini ke area deploy.
3. Netlify akan memberikan URL langsung. Bisa hubungkan domain custom.

### Hostinger / Shared Hosting
1. Upload file ke folder `public_html` lewat File Manager atau FTP.
2. Arahkan domain ke hosting.

## Setelah live
- Ganti `sitemap.xml` loc ke domainmu, lalu submit di Google Search Console.
- Tambahkan meta tags og:image (untuk preview sosial) dan update robots.txt jika perlu.
- Ganti `cover-placeholder.jpg` dengan cover final.

Butuh gua uploadin langsung ke GitHub/Netlify? Bilang aja — gua siap bantu sampai live.
